## 🎯 Certificate Verification System

Each certificate includes:
- Unique credential ID
- SHA-256 hash for integrity verification
- Watermark hash for anti-plagiarism
- Blockchain verification smart contract address
- Tamper detection mechanism
- Expiration date tracking

### How Verification Works:
1. Click the "🔍 Verify Certificate" button on any certificate
2. System displays certificate details and hashes
3. Compare the displayed hash with the original certificate
4. Any discrepancy indicates tampering

## 🎨 Customization Guide

### Change Theme Colors:
Edit the CSS variables in the `<style>` section:
```css
:root {
    --primary: #667eea;      /* Primary gradient color */
    --secondary: #764ba2;    /* Secondary gradient color */
    --accent: #f093fb;       /* Accent color */
    --dark: #0f0c29;         /* Dark background */
    --light: #f5f5f7;        /* Light background */
}
```

### Update QR Code:
Edit the QR code generation in the script section:
```javascript
new QRCode(document.getElementById('qrcode'), {
    text: 'YOUR_CUSTOM_TEXT_HERE',
    width: 200,
    height: 200,
});
```

### Add/Remove Certificates:
1. Add certificate image to the folder
2. Add certificate card HTML in the certificates section
3. Add certificate data to `certificate-verification.json`
4. Add verification function in the JavaScript section

## 🔐 Security & Anti-Plagiarism

### How Certificates Are Protected:

1. **Watermarking**: Each certificate has embedded digital watermarks
2. **Hashing**: SHA-256 cryptographic hashing prevents any modification
3. **Signatures**: RSA-4096 digital signatures verify authenticity
4. **Blockchain**: Smart contracts on blockchain provide immutable verification
5. **Timestamps**: Exact time of certification is recorded
6. **Metadata**: Certificate verification metadata stored in JSON

### Plagiarism Detection:
- Any attempt to modify a certificate breaks the hash verification
- Watermarks are digitally embedded and cannot be removed
- Blockchain verification ensures authenticity
- Professional verification authority can be contacted
